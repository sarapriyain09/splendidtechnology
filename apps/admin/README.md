# Velynxia Admin Console

Admin is a shared platform service used by all Velynxia product platforms.

It is not treated as an independent product line.

Planned responsibilities:
- Authentication and access governance
- Role and permission management (RBAC)
- Tenant and workspace administration
- Licensing and entitlement controls
- Platform-wide settings and policies
- Audit and governance controls

Status: planning (foundation service placeholder initialized).
