- CTA: Try CRM Demo or Book a CRM Demo

- CTA (10-15 seconds)

## Decision rules for next 90 days

- If content topic does not clearly support CRM or AI automation demand, postpone it
- If post idea cannot link to a core service or demo CTA, rewrite it
- If a topic generates high engagement but low demo intent, publish fewer variants
- If a topic drives demos, produce one follow-up post and one video within 7 days
