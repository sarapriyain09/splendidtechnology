# Velynxia Development Roadmap

## Professional Portfolio Model

Velynxia is managed as one company roadmap with shared platform services plus five platforms.

Shared Platform Services (Phase 0 foundation)
- Authentication, PostgreSQL, Prisma, licensing, notifications, audit, storage, API gateway, shared UI/reporting/settings

Five platforms
1. Growth Platform
2. AI Media Suite
3. Commerce Platform
4. Engineering Platform
5. Administration (shared management layer, not an independent market product)

## Strategic Principle

Admin remains a shared platform capability used by all platforms through consistent governance, security, and operations contracts.
